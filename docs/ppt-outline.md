> **전체 스토리**: 단순한 방법부터 시작해, 각 단계의 한계를 발견하고 보완해나가며 최종 모델(A4)에 도달하는 과정

## 슬라이드 구성 (총 10슬라이드)

---

### 슬라이드 1 — 연구 질문

**스토리**: 왜 이 연구가 필요한가

**제목**: 최소 비교 횟수로 선호도를 얼마나 정확히 학습할 수 있는가?

**내용**:
- 부동산 추천에서 사용자가 참여하는 비교 횟수는 제한적
- 목표: 숨겨진 선호 가중치 `w*` (20차원)를 최소 비교로 복원
- **측정 지표**: 학습 가중치와 `w*`의 코사인 유사도 ≥ 0.9 달성에 걸리는 비교 횟수

**만들 자료**: PPT 도형으로 흐름도
```
[사용자: 매물 A vs B 비교] → [MCMC 사후 갱신] → [선호 가중치 추정] → [코사인 유사도 측정]
```

**논문**: Chu & Ghahramani (2005)

---

### 슬라이드 2 — 실험 설계: 3가지 접근법

**스토리**: 크게 3가지 방법으로 나누어 실험했다

**제목**: 실험 설계 — 3단계 점진적 개선

**내용**: 3개 브랜치 박스 (PPT 도형)

```
Branch 1               Branch 2                Branch 3
Flat Bayesian    →    Simple Hierarchy    →    Active Query Hierarchy
  (F1~F4)               (H1~H4)                  (A1~A4 = 우리 모델)

단일 레벨 MCMC        2단계 계층 구조           능동 쿼리 + 도메인 제약
20D 공간 직접 학습    매크로 + 마이크로           가상 아이템 + 지리 보너스
```

**만들 자료**: 파랑→초록→빨강 박스 3개, 화살표 연결 (PPT 도형)

---

### 슬라이드 3 — Branch 1: Flat 모델과 한계

**스토리**: 가장 기본적인 방법 → 성능 확인 → 한계 발견

**제목**: Branch 1 — 단일 레벨 베이지안 학습 (F1~F4)

**내용 ①**: 단계별 개선 표

| 변형 | 추가한 것 | 도달률 | 평균 라운드 | 최대 코사인 |
|------|-----------|--------|------------|------------|
| F1 | 없음 (랜덤 쌍) | **0%** | 미달성 | 0.755 |
| F2 | EVR 쿼리 선택 | 10% | 176.5회 | 0.825 |
| F3 | + 사전 분포 초기화 | 5% | 90.0회 | 0.833 |
| F4 | + BWM 초기화 | **0%** | 미달성 | 0.828 |

**내용 ②**: 한계 3가지 (텍스트 박스)
- ① 랜덤 쌍(F1): 정보 없는 비교 → 0% 도달
- ② EVR 도입(F2)해도 176회 필요 → **비현실적**
- ③ 근본 원인: **20D 전체를 한꺼번에 학습** → 차원의 저주

**만들 자료**: 위 표를 PPT 표로 제작 (F2 행을 파란 배경으로 강조)

**논문**: Chu & Ghahramani (2005), Brochu et al. (2007), Rezaei (2015)

---

### 슬라이드 4 — Branch 2: 계층 구조 도입

**스토리**: Flat의 한계를 계층 구조로 극복한다

**제목**: Branch 2 — 2단계 계층 학습 도입 (H1~H4)

**내용 ①**: 계층 구조 다이어그램 (PPT 도형)
```
1단계 — 매크로 학습 (4D simplex)
   어떤 카테고리를 중시하는가?
   [거리] [가격] [안전] [편의성]
         ↓ 상위 2개 카테고리 선택
2단계 — 마이크로 학습 (서브 차원)
   그 카테고리 안에서 무엇을 중시하는가?
   거리: [도보 vs 버스 vs 경사도]
   가격: [월세 vs 보증금 vs 관리비]
```

**내용 ②**: 단계별 개선 표

| 변형 | 추가한 것 | 도달률 | 평균 라운드 | 최대 코사인 |
|------|-----------|--------|------------|------------|
| H1 | 계층 구조 (단위 구) | **0%** | 미달성 | 0.669 |
| H2 | + 심플렉스 투영 | 10% | 22.5회 | 0.831 |
| H3 | + AMPLe γ=0.7 | 5% | 17.0회 | 0.826 |
| H4 | + 가상 아이템 | **25%** | **7.8회** | **0.869** |

**강조**: H2 시점에서 176회 → 22회로 **8배 감소**, H4에서 7.8회로 추가 감소

**만들 자료**: 위 표를 PPT 표로 제작 (H4 열을 초록 배경으로 강조)

**논문**: Fürnkranz & Hüllermeier (2010), Duchi et al. (2008), Oh et al. (2019)

---

### 슬라이드 5 — 가상 아이템 효과 (집중 분석)

**스토리**: H3→H4의 핵심인 가상 아이템이 실제로 얼마나 기여하는가

**제목**: 가상 아이템 도입 효과 — 전용 실험 (각 50회 반복)

**내용 ①**: 수렴 곡선 SVG 삽입
- 파일: `out/simulation/virtual-comparison-curve.svg`
- 회색(H3 없음) / 초록(H4 단순) / 빨강(A3 Trade-off) 평균 곡선
- PPT → 삽입 → 그림으로 바로 삽입 가능

**내용 ②**: 결과 비교 표 (SVG 오른쪽에 배치)

| | H3 (없음) | H4 (단순) | A3 (Trade-off) |
|--|-----------|-----------|----------------|
| 도달률 | 14% | **20%** | 18% |
| 평균 라운드 | 9.3회 | 11.6회 | 15.4회 |
| 최대 코사인 | 0.841 | **0.859** | 0.852 |

**강조 문구**:
> "가상 아이템 도입으로 목표 도달률 **+43% 향상** (14% → 20%)"
> "최대 코사인 유사도 0.841 → 0.859 (+2.1%)"

**가상 아이템이란?**: 도형 박스 1개 추가
```
실제 매물로는 만들 수 없는 "극단적인 비교 쌍" 합성
  아이템 A: 거리=0.95, 가격=0.05 (가깝지만 비쌈)
  아이템 B: 거리=0.05, 가격=0.95 (멀지만 저렴함)
  → 거리 vs 가격 중 무엇이 더 중요한지 직접 파악
```

**만들 자료**:
1. `virtual-comparison-curve.svg` PPT에 삽입 (슬라이드 왼쪽 60%)
2. 결과 표 PPT 표로 직접 제작 (슬라이드 오른쪽 40%)
3. 도형 박스 1개 (슬라이드 하단)

**논문**: Sadigh et al. (2017)

### 슬라이드 6 — Branch 3: 우리 최종 모델

**스토리**: H4에서 3가지를 더 보완해 최종 모델 완성

**제목**: Branch 3 — 능동 쿼리 계층 모델 (A1~A4)

**내용 ①**: 단계별 개선 표

| 변형 | 추가한 것 | 도달률 | 평균 라운드 | 최대 코사인 |
|------|-----------|--------|------------|------------|
| A1 | Trade-off 가상 쌍 (Sadigh 2017) | 5% | 4.0회 | 0.847 |
| A2 | + BWM 마이크로 사전 초기화 | 15% | 5.7회 | 0.849 |
| A3 | + Ambiguity 기반 쌍 선택 | 20% | 6.5회 | 0.864 |
| **A4** | + **지리적 보너스** (도메인 제약) | **25%** | 14.6회 | 0.861 |

**내용 ②**: 각 추가 요소 한 줄 설명 (텍스트 박스 3개)
- **Trade-off 쌍**: C(4,2)=6쌍, 두 카테고리 직접 대결 → 차이벡터 1.8배 커짐
- **BWM 초기화**: 중요도 비율 사전 추정 → cold start 완화 (Rezaei 2015)
- **Ambiguity 선택**: P(A≻B)≈0.5인 쌍 우선 → 정보 획득 극대화

**만들 자료**: 위 표를 PPT 표로 제작 (A4 열을 빨강 배경으로 강조)

**논문**: Sadigh et al. (2017), Rezaei (2015)

### 슬라이드 9 — 각 구성 요소 기여도

**스토리**: 우리 모델의 각 요소가 실제로 기여하는가

**제목**: 구성 요소별 기여도 (Ablation)

**내용**: 논문 근거 포함 기여도 표

| 기법 | 개선 효과 | 논문 |
|------|-----------|------|
| 심플렉스 투영 | H1(0%) → H2(10%), 176회 → 22회 | Duchi et al. (2008) |
| AMPLe γ=0.7 | 22회 → 17회 | Oh et al. (2019) |
| **가상 아이템** | **5% → 25%, 17회 → 7.8회** | Sadigh et al. (2017) |
| BWM 초기화 | 도달률 5% → 15% | Rezaei (2015) |
| Ambiguity 선택 | 도달률 15% → 20% | Sadigh (2017) §4 |
| 지리적 보너스 | 편향 제거, 도달률 유지 | 자체 기여 |

**강조**: 가상 아이템 행을 굵게/배경색으로 강조 → 가장 큰 단일 기여

**만들 자료**: PPT 표로 직접 제작

### 슬라이드 10 — 결론

**스토리**: 한 줄로 요약

**제목**: 결론 — 계층 구조 + 가상 아이템 = 수렴 최적화

**내용 ①**: 핵심 결과 3줄
```
1. 계층 구조 도입만으로 176회 → 22회 (8배 감소)
2. 가상 아이템 추가로 22회 → 7.8회 (3배 추가 감소)
3. 전체: F2(176회, 10%) → H4(7.8회, 25%) = 22.6배 빠름 + 2.5배 높은 도달률
```

**내용 ②**: 한계 및 향후 과제
- 현재 도달률 25% → 향후 가상 아이템 다양화로 개선 가능
- 실제 사용자 실험 필요 (시뮬레이션 oracle → 실제 응답)

**만들 자료**: 없음 (텍스트 + 강조 박스만)

